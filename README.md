# タイピングクエスト

日本語のお題をローマ字で入力したり、1文字ずつキー位置を練習したりできるHTMLタイピングゲームです。

GitHub Pagesで公開できる静的HTMLゲームです。サーバーやデータベースは不要で、`index.html` をブラウザで開くだけで遊べます。

ハイスコアは各端末のブラウザ内に保存されます。ランキング共有やオンライン保存はまだありません。

## モード

### 文章タイピングモード

漢字かな交じりの日本語フレーズを見て、ローマ字で入力します。画面には `displayText` を表示し、ローマ字判定には別に持たせたひらがなの `reading` を使います。

入力が正しく進むごとにオリジナルモンスターのHPが減り、1問を打ち終えるとモンスターが倒されて次のお題に進みます。モンスターは弱いスライム風の敵から中ボス風の敵まで段階的に登場します。倒したモンスター数は結果画面に表示されます。

お題は最初は1〜3文字程度の短いものから始まり、後半に向けて少しずつ長くなります。

正解が連続するとコンボ攻撃が表示されます。10問ごとに中ボスが登場し、結果画面では倒したモンスターを図鑑のように確認できます。

ミスが多かったローマ字パターンも結果画面に表示されます。

ヘボン式と一般的な日本語入力式の両方に対応しています。`shi / si`、`chu / tyu / cyu`、`n / nn`、`ou / oo` などの入力ゆれを受け付けます。

通常の「え」は `e` として扱い、`ye` にはしません。`ye` は「いぇ」のような特殊な拗音だけで候補になります。

### 一文字キーボード練習モード

画面に表示された1文字の英字キーを押す練習モードです。PCの物理キーボードでホームポジションを覚える練習に向いています。

簡易QWERTYキーボードを表示し、押すべきキーをハイライトします。ホームポジションの `A S D F / J K L ;` がわかりやすく表示されます。

一文字モードでは文章モード用のお題、ローマ字入力欄、モンスター表示、倒した数などは表示されません。

出題は `a` 〜 `z` と `－` のキーからランダムに選ばれます。ホーム段、上段、下段がバランスよく出るようにし、同じキーが連続しすぎないようにしています。ミスが多かった苦手キーは、後半で少しだけ再出題されやすくなります。

練習タイプは「全体」「ホーム」「左手」「右手」から選べます。

## 遊び方

1. `index.html` をブラウザで開きます。
2. 最初の画面でモードを選びます。
3. スタートボタン、またはEnterキーで開始します。
4. 文章モードではローマ字を入力します。
5. 一文字モードでは表示されたキーを物理キーボードで押します。
6. 60秒後に結果画面が表示されます。

## GitHub Pagesで公開する方法

1. このフォルダ内の `index.html`、`style.css`、`script.js`、`README.md` をGitHubリポジトリにアップロードします。
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. `Pages` を開き、公開元のブランチを選びます。
4. ルートに `index.html` を置いている場合は、公開フォルダを `/root` にします。
5. 表示されたGitHub PagesのURLを開いて動作確認します。

すべての読み込みは相対パスなので、GitHub Pages上でもそのまま動きます。

## 結果画面

- 正解数
- ミス数
- 正確率
- スコア
- ランク `S / A / B / C`
- 倒した敵の数
- 苦手だったキー
- 苦手だったローマ字
- 最高スコア

文章タイピングモードと一文字キーボード練習モードの最高スコアは別々に保存されます。

## ファイル構成

- `index.html`: 画面構造
- `style.css`: デザイン
- `script.js`: ゲーム処理
- `assets/monsters/monster-01.png` 〜 `monster-10.png`: 文章モードで表示するオリジナルモンスター画像
- `README.md`: 説明

## 拡張しやすいポイント

- `PHRASE_DATA` にお題を追加できます。
- 必要な例外だけ `extraInputs` に追加できます。
- `displayText` は画面表示、`reading` はローマ字候補生成用です。
- `KEY_POOL_WEIGHTS` を変えると、一文字モードの出題バランスを調整できます。
- `GAME_SETTINGS.duration` を変えると制限時間を変更できます。
